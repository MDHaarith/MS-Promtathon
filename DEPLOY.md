## ‚≠ê Option 1: Netlify (Recommended - Easiest)

### Step 1: Connect to Netlify
1. Go to [netlify.com](https://netlify.com)
2. Click **"New site from Git"**
3. Select **GitHub** as your provider
4. Choose your **MS-Promtathon** repository
5. Netlify auto-detects settings from `netlify.toml`
6. Click **Deploy Site**

### Step 2: Wait for Deployment
- First deploy takes 2-3 minutes
- You get a URL like: `https://xxxxx.netlify.app`

### Step 3: (Optional) Auto-Deploy on GitHub Push
To auto-deploy every time you push to `main`:

1. **Get Netlify tokens:**
   - Go to [Netlify Settings > Applications](https://app.netlify.com/user/applications/personal)
   - Create **Personal Access Token**, copy it

2. **Get Site ID:**
   - Go to your Netlify site > **Settings > General**
   - Copy **Site ID**

3. **Add GitHub Secrets:**
   - Go to your GitHub repo
   - Settings > **Secrets and Variables > Actions**
   - Add `NETLIFY_AUTH_TOKEN` = (token from step 1)
   - Add `NETLIFY_SITE_ID` = (ID from step 2)

4. **Done!** Push to main ‚Üí Netlify auto-deploys

**Cost:** Free tier includes 300 build minutes/month  
**Your Site URL:** https://YOUR-SITE.netlify.app

## üåê Option 2: Railway.app (Simple Full-Stack)

### Step 1: Create Railway Account
1. Go to [railway.app](https://railway.app)
2. Sign up with GitHub
3. Click **New Project**

### Step 2: Deploy
1. Select **Deploy from GitHub repo**
2. Choose your **MS-Promtathon** repository
3. Railway auto-detects it's a Node.js app
4. Click **Deploy**

### Step 3: Set Environment
- Go to project **Variables**
- Set `NODE_ENV=production`
- Railway auto-assigns a domain

**Cost:** Free tier with $5/month credit  
**Your Site URL:** Auto-generated by Railway

---

## üî• Option 3: Render.com (No Credit Card Required)

### Step 1: Create Account
1. Go to [render.com](https://render.com)
2. Sign up with GitHub
3. Click **New +**

### Step 2: Create Web Service
1. Select **Web Service**
2. Choose your GitHub repo
3. Configure:
   - **Name:** `ms-promtathon`
   - **Environment:** `Node`
   - **Build Command:** `pnpm build`
   - **Start Command:** `pnpm start`
4. Click **Create Web Service**

**Cost:** Free tier available  
**Your Site URL:** `https://ms-promtathon.onrender.com`

## üê≥ Option 4: Docker + Any Cloud (Advanced)

If you want full control, use Docker:

# Run locally to test

# Push to Docker Hub

# Then deploy to: AWS, Google Cloud, DigitalOcean, Azure, etc.

## üö® Troubleshooting

### Site shows blank page
- Check browser console (F12) for errors
- Run `pnpm build` locally first
- Verify `.env` variables are set in platform

### Build fails
- Check build logs in platform dashboard
- Ensure `pnpm install` succeeds
- Verify Node.js version (20.x or 22.x)

### API endpoints not working
- Check that `server/` files built correctly
- Verify platform supports Node.js backends
